If age is 18 or older, return "You are eligible to vote."

Check Temperature and Suggest Clothing:

Create a function named suggestClothing that takes a single parameter temperature (in degrees Celsius) and returns a message based on the following conditions:

If temperature is less than 0, return "It's freezing. Wear a heavy coat and boots."
If temperature is between 0 and 15 (inclusive), return "It's cold. Wear a jacket."
If temperature is between 16 and 25 (inclusive), return "It's mild. A sweater should be fine."
If temperature is above 25, return "It's warm. Wear light clothing."
Grade Evaluation:

Create a function named evaluateGrade that takes a single parameter score (a number between 0 and 100) and returns a letter grade based on the following conditions:

If score is 90 or higher, return "A"
If score is between 80 and 89 (inclusive), return "B"
If score is between 70 and 79 (inclusive), return "C"
If score is between 60 and 69 (inclusive), return "D"
If score is below 60, return "F"
